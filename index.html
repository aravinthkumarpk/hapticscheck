Custom Haptics

Create custom vibration patterns by entering a sequence of vibration durations separated by commas.

Example: 200, 100, 200, 100, 200

  • Each number represents the vibration duration in milliseconds.
  • Odd-indexed numbers (1st, 3rd, 5th, etc.) represent vibration durations.
  • Even-indexed numbers (2nd, 4th, 6th, etc.) represent pause durations between vibrations.

For more information, refer to the Vibration API documentation.
